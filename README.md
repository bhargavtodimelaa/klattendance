# 🎓 KL University Attendance Calculator

🔗 **Live Project:**  
https://klattendance-alpha.vercel.app

A fast and simple **Attendance & Bunk Calculator** built for **KL University students**.

This tool helps students quickly check their attendance and determine how many classes they can safely bunk or need to attend to maintain the required attendance percentage.

---

# ✨ Features

- ✔ Attendance percentage calculation
- ✔ Safe bunk calculator
- ✔ Shows required classes to maintain attendance
- ✔ Supports **L-T-P-S academic structure**
- ✔ Fast and lightweight
- ✔ Works on **mobile and desktop**
- ✔ Clean minimal interface

---

# 📊 What It Calculates

The tool automatically calculates:

- Current Attendance Percentage
- Safe Bunk Count
- Classes Required to Maintain Attendance
- Attendance based on **L-T-P-S structure**

---

# 📚 L-T-P-S Structure

| Type | Meaning |
|-----|--------|
| **L** | Lecture |
| **T** | Tutorial |
| **P** | Practical |
| **S** | Skill / Seminar |

# ⚙️ How It Works

1. Login using your **KL University ERP credentials**
2. Enter your **ERP username and password**
3. The system securely fetches your **attendance data from the ERP portal**
4. The calculator automatically shows:

- Attendance Percentage
- Safe Bunk Count
- Classes Required to Maintain Attendance

Everything updates instantly after fetching the data using **fast JavaScript processing**.
