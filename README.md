# 📊 KL Attendance — Academic Intelligence Dashboard

A powerful, modern web dashboard for Kl university students that helps students analyze, track, and optimize their attendance with precision.

---

## 🚀 Features

- 🔐 **Secure Login System**
  - Student ID + Password authentication
  - CAPTCHA verification support

- 📈 **Attendance Analytics**
  - Real-time attendance percentage calculation
  - Weighted component-based evaluation:
    - Lecture (L)
    - Practical (P)
    - Tutorial (T)
    - Skilling (S)
  - Visual breakdown per subject

- 🧠 **Smart Insights**
  - ✅ Safe (≥85%)
  - ⚠ Conditional (75–84%)
  - ❌ Danger (<75%)

- 📅 **Class Planning**
  - Maximum classes you can skip
  - Minimum classes you must attend
  - Smart projections based on weights

- 🧮 **Next Class Simulator**
  - Simulate future attendance
  - Add classes (Single / Double / Triple)
  - Choose Attend / Skip
  - Instant recalculation

- 🔍 **Search & Filters**
  - Search subjects
  - Filter by status

- 📤 **Export**
  - CSV export
  - Print support

- 🎨 **Modern UI**
  - Dark theme
  - Smooth animations
  - Fully responsive
